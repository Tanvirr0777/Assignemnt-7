Name of project : Friendship-Infiity

Friendship infinity is a platform where you can create a good connection to your friend, you can communicate with your friend using call,text,video. 


Technologies used: 
                 React.js ,
                 tailwindCSS ,
                 daisyui


Key features:
            track on how you communicate with your friends ,
            shows communication medium in a chart with compare ,
            user-friendly user interface
